# The start
Before start you need to install all the dependencies.<br/>
So copy and paste this command in your terminal :<br/>

with **npm**<br/>
``` cd shitty_react && npm install && npm start``` <br/>
with **yarn**<br/>
``` cd shitty_react && yarn install && yarn start```<br/>
